welcome to hidn.art

hidn.art is a website designed using HTML-5, CSS, and Javascript as part of the Code First Girls Introduction to Web Development Kickstarter.

This site exists mostly for demonstrative purposes to showcase coding skills learned, but also looks at the idea of things 'hidden' within visual art, like the hidden process behind finished pieces, and hidden illusions in seemingly face-value artworks.
